# Code of Conduct

## Our pledge

We want the Smart Agents BC SDK community to be welcoming, productive, and respectful for everyone, regardless of experience level, background, identity, or affiliation.

## Expectations

Participants are expected to:

- Be considerate and constructive in all interactions (issues, pull requests, discussions, code reviews).
- Assume good faith — most disagreements come from missing context, not bad intent.
- Focus feedback on technical content, not on the person.
- Respect differences in opinion; surface disagreements openly and politely.
- Help newcomers and answer questions where you can.

Behavior that makes the community less welcoming — personal attacks, deliberate intimidation, sustained disruption, or unwelcome contact toward another participant — is not acceptable and will be addressed by the maintainers.

## Scope

This Code of Conduct applies to all project spaces: the GitHub repository (issues, pull requests, discussions, commits, code review comments), and any public space where someone is representing the project (talks, social media posts, online events).

## Reporting

If you witness or experience behavior that violates this Code of Conduct, please contact the maintainers privately at **conduct@qualia-technik.de**.

Reports will be handled discreetly. You can expect:

- Acknowledgement within 5 business days.
- A summary of how the report will be reviewed.
- A response on actions taken, where appropriate.

## Enforcement

Maintainers may take any action they consider proportionate, including a private warning, removing a comment, blocking a participant from the repository, or escalating to GitHub. Repeated or severe violations will result in a permanent ban.
